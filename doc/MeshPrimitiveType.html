MeshPrimitiveType

Describes the type of primitives stored in a mesh object.

Choices

Number Name Description
0 MESH_PRIMITIVE_TRIANGLES Triangle primitive type.
1 MESH_PRIMITIVE_TRISTRIPS Triangle strip primitive type.
2 MESH_PRIMITIVE_LINES Lines primitive type.
3 MESH_PRIMITIVE_LINESTRIPS Line strip primitive type.
4 MESH_PRIMITIVE_QUADS Quadrilateral primitive type.
5 MESH_PRIMITIVE_POINTS Point primitive type.

Storage Type

uint
