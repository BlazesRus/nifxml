ConstraintPriority

hkpConstraintInstance::ConstraintPriority. Priority used for the constraint.
Values 2, 4, and 5 are unused or internal use only.

Choices

Number Name Description
0 PRIORITY_INVALID PRIORITY_INVALID
1 PRIORITY_PSI Constraint is only solved at regular physics time steps.
3 PRIORITY_TOI Constraint is also solved at time of impact events.
