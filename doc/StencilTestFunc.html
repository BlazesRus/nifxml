StencilTestFunc

Describes stencil buffer test modes for NiStencilProperty.

Choices

Number Name Description
0 STENCIL_TEST_NEVER Always false. Ref value is ignored.
1 STENCIL_TEST_LESS VRef ‹ VBuf
2 STENCIL_TEST_EQUAL VRef = VBuf
3 STENCIL_TEST_LESS_EQUAL VRef ≤ VBuf
4 STENCIL_TEST_GREATER VRef › VBuf
5 STENCIL_TEST_NOT_EQUAL VRef ≠ VBuf
6 STENCIL_TEST_GREATER_EQUAL VRef ≥ VBuf
7 STENCIL_TEST_ALWAYS Always true. Buffer is ignored.

Storage Type

uint
