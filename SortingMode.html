SortingMode

Describes the way that NiSortAdjustNode modifies the sorting behavior for the subtree below it.

Choices

Number Name Description
0 SORTING_INHERIT Inherit. Acts identical to NiNode.
1 SORTING_OFF Disables sort on all geometry under this node.

Storage Type

uint
